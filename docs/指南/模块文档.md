## utils
工具模块，提供一些工具方法
### isNative
`() => boolean`
判断宿主是否为客户端，在打包后的 App 中 isNative 返回 true，其它场景下返回 false
```js
const utils = require('utils');

if (utils.isNative()) {
   // do something
} else {
   // do something
}
```
### getWidgetImageUrl
`(str) => str`
输入素材库文件名，返回素材库文件链接
```js
const utils = require('utils');

imageUrl = utils.getWidgetImageUrl('LOGO.jpg')
```
