# ILQ ( I Love Qii )
::: tip 作者
`@Inventocode`
:::

什么，原版控件不好用？来逝世这个“狸猫换太子”脚本吧！

功能：将所有可替换的原生控件替换为Qii控件

需要提前导入Qii控件，建议搭配[CCWMP](/实用工具/CCWMP)使用
## 安装

::: tip [ILOVEQII.user.js](https://static.codemao.cn/flowchunkflex/H1pvAd-Sgl.js)
好用好用！
:::

## 使用

1. 先导入Qii控件库1.0及2.0
2. 按照正常方式导入控件即可
