# CCQI (CoCo Quick Import)

::: tip 作者
原作者：`@Inventocode`

优化：`@小宏`
:::

这个工具可以帮助你快速导入自定义控件

## 安装

[CCQI_小宏优化版本.js](https://static.codemao.cn/flowchunkflex/SkVg5yvble.js)

1. 确认你的浏览器有一个用户脚本管理器
2. 把工具源码拖动至浏览器
3. 管理器应该会捕获它，选择安装

## 使用

1. 在文件资源管理器中找到你的自定义控件文件
2. 打开CoCo编辑器
3. 把文件拖拽到“顶部菜单-文件”图标上，应该会显示一个遮罩
4. 松开鼠标，控件将被导入
