## 使用

1. 先打包你的作品
2. 打包完成后点击“Omnia改包”按钮
3. 等待改包完成（需要1~3分钟）
4. 弹出新的窗口，可以下载、复制链接或手机扫码
5. 窗口关闭后可以通过右侧悬浮按钮再次打开
